# 원장 스타일 5유형 진단 시스템

치과/병원 원장의 운영 스타일을 5가지 유형으로 진단하고 맞춤형 컨설팅 전략을 제안하는 웹 애플리케이션입니다.

## 📋 프로젝트 개요

- **목적**: B2B 컨설팅 영업 도구 / 프랜차이즈 사업 지원
- **대상**: 치과/병원 원장
- **소요 시간**: 5~7분 (25문항)
- **결과**: 주 유형 + 보조 유형 + 맞춤 전략 로드맵

## 🎯 5가지 원장 유형

1. **결단·속도형** (A형) - 빠른 실행과 즉각적 효과 중시
2. **구조·안정형** (B형) - 기준과 시스템화 중시
3. **팀·교육형** (C형) - 직원 성장과 조직 안정 중시
4. **수익·경영형** (D형) - ROI와 KPI 중심
5. **환자·신뢰형** (E형) - 환자 경험과 브랜드 이미지 중시

## 🚀 사용 방법

### 1. 실행하기

브라우저에서 `index.html` 파일을 열어주세요.

**Windows:**
```
더블클릭으로 index.html 실행
또는
마우스 우클릭 > 연결 프로그램 > 브라우저 선택
```

**로컬 서버 실행 (권장):**
```bash
# Python이 설치되어 있다면
python -m http.server 8000

# 브라우저에서 접속
http://localhost:8000
```

### 2. 진단 진행

1. **시작 페이지**: 병원명과 원장명 입력
2. **설문 페이지**: 25개 질문에 답변
   - ○ (1순위): 가장 가까운 답변 선택
   - △ (2순위): 두 번째로 가까운 답변 선택
3. **결과 페이지**: 진단 결과 및 맞춤 전략 확인

### 3. 답변 방식

각 질문마다:
- **1~5번** 선택지 중
- **2개**를 선택 (1순위 + 2순위)
- 예시: "2, 5" → 2번이 1순위, 5번이 2순위

## 📊 점수 계산 방식

- **1순위 선택**: 2점
- **2순위 선택**: 1점
- **25문항 합산**: 총 75점 만점
- **주 유형**: 가장 높은 점수
- **보조 유형**: 두 번째로 높은 점수

## 📈 테스트 데이터

**화이트 서울 치과 | 임영일 원장 (예시)**

- **지역**: 대전 중구
- **의사**: 1명
- **간호사**: 1명
- **상담사**: 1명

```
Q1: 2,5  Q2: 4,5  Q3: 2,3  Q4: 5,1  Q5: 5,3
Q6: 5,2  Q7: 4,3  Q8: 4,5  Q9: 2,3  Q10: 4,2
Q11: 4,5  Q12: 1,5  Q13: 4,5  Q14: 4,5  Q15: 2,3
Q16: 1,4  Q17: 1,4  Q18: 4,5  Q19: 2,5  Q20: 2,5
Q21: 1,4  Q22: 2,4  Q23: 4,2  Q24: 4,5  Q25: 1,5
```

**예상 결과:**
- 주 유형: 4형 (수익·경영형) - 24점
- 보조 유형: 5형 (환자·신뢰형) - 18점

## 🎨 주요 기능

### ✅ 완성된 기능

1. **상단 네비게이션 메뉴** - 모든 페이지에서 자유롭게 이동 가능 (반응형 햄버거 메뉴)
2. **병원 정보 입력** - 병원명, 원장명, 지역, 의사 수, 간호사 수, 상담사 수
3. **25문항 설문** - 1순위/2순위 선택
4. **실시간 진행률** - 프로그레스 바
5. **답변 수정** - 이전 질문으로 돌아가기
6. **자동 분석** - 5가지 유형 점수 집계
7. **레이더 차트 시각화** - 5가지 유형 점수를 방사형 차트로 표시
8. **맞춤 리포트** - 주/보조 유형 분석
9. **전략 로드맵** - 유형별 4단계 실행 계획
10. **시연 모드** - 임영일 원장 데이터 자동 입력 데모
11. **인쇄 기능** - 결과 출력

### 🔜 향후 개발 예정

1. **PDF 출력** - 전문적인 리포트 생성
2. **데이터 저장** - 진단 이력 관리
3. **관리자 페이지** - 통계 및 분석
4. **이메일 발송** - 자동 리포트 전송

## 🛠 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
